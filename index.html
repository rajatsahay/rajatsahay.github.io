Dynamic Template Selection through Change Detection for Adaptive Siamese Tracking
Madhu Kiran, Le Thanh Nguyen-Meidine, Rajat Sahay, Rafael Menelau-Cruz, Louis-Antoine Blais-Morin, Eric Granger

Under Review
arXiv/code

We formulate Single Object Tracking (SOT) as an online incremental learning problem and formulate a new method for dynamic sample selection and memory replay, preventing template corruption. In particular, we propose a change detection mechanism to detect gradual changes in object appearance and select the corresponding samples for online adaptation. We also introduce an entropy-based sampling strategy to maintain a diversified auxiliary buffer for memory replay. Our proposed method can be integrated into any object tracking algorithm that leverages online learning for model adaptation.

Generative Target Update for Adaptive Siamese Tracking
Madhu Kiran, Le Thanh Nguyen-Meidine, Rajat Sahay, Rafael Menelau-Cruz, Louis-Antoine Blais-Morin, Eric Granger

Accepted to ICPRAI 2022, paper up soon.
arXiv/project/code

We propose a model adaptation method for Siamese trackers that use a generative model to produce a synthetic template from the object search regions of several previous frames, rather than directly using the tracker output. Since the search region encompasses the target, we introduce an attention-based mechanism for robust model adaptation. In particular, our approach relies on an auto-encoder trained through adversarial learning to detect changes in a target object's appearance and predict a future target template, using a set of target templates localized from tracker outputs at previous frames. To prevent template corruption during the update, the proposed tracker also performs change detection using the generative model to suspend updates until the tracker stabilizes, and robust matching can resume through dynamic template fusion.

Graph Segmentation in Scientific Datasets
Rajat Sahay, Savannah Thais

NeurIPS (Workshop) 2021
paper/poster

We study the impact of graph segmentation on two tasks: mapping the trajectories of charged particles at the Large Hadron Collider (LHC) and segmenting bindable ligand sites in proteins. We explore graph segmentation as a precursor to a larger deep learning pipeline. By understanding more about the underlying geometric structure of the data and, when possible, separating the input data into distinct sub-graphs, we are able to simplify the tasks of downstream graph-based deep learning architectures and parallelize inference in constrained computing environments.

An Enhanced Prototypical Network Architecture for Few-Shot Handwritten Urdu Character Recognition

Under Review, Pattern Recognition
arXiv (Preprint up soon!)

Unrestricted Adversarial Attacks on Vision Transformers
Rajat Sahay

CVPR (Workshop) 2021
paper/arXiv/poster

We leverage colorization models to semantically attack images generating low-frequency adversarial perturbations while changing colors in ambigous areas. We were able to demonstrate that this method was successful in getting Vision Transformers (ViTs) to misclassify image labels.
Visual Odometry Problems in Constrained Environments
Rajat Sahay, Surya Prakash
[Project Report]

Leveraged optical flow methods to map trajectories of random, fast-moving particles in constrained environments. Used statistical methods like Lucas-Kanade Optical Flow combined with Hungarian Algorithm to track and predict feature frames based on inputs by previous frames.
